# Websocket在线聊天室

这是一个基于Vue3和Websocket实时通讯技术以及Element Plus构建的前端网页在线聊天室，旨在使用最少的代码和最简单的逻辑来模拟在线聊天环境。

## 运行项目

```sh
npm install
npm run dev
```

## 运行后端

```sh
cd server
npm install
cd ../
npm run server
```

> 注意：后端需要使用Node.js运行，并且需要安装 `ws` 和 `express` 这两个Node.js包。

## 打包项目

```sh
npm run build
```

## 清理记录

```sh
npm run clear
```
